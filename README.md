Oral-Cancer-Detection

AI-based system for early detection of oral cancer using deep learning.
---
🚀 Features
- Multi-dataset training
- EfficientNet B0, B1, B2 baselines
- Ensemble learning (B1 + B2)
- Cross-dataset evaluation
- False negative tracking (medical focus)
- Streamlit interactive app
---
📊 Models

| Model | Description |
|------|------------|
| B0 | Baseline |
| B1 | Improved accuracy |
| B2 | Higher resolution |
| Ensemble | Best performance |
---
⚠️ Medical Insight

Minimizing **false negatives** is critical in cancer detection.
---
🖥 Run App

pip install -r requirements.txt  
streamlit run app/app.py
